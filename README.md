# AI Agents Bootcamp (7-Day Crash Course)

This repository contains my implementation of the **AI Agents Crash Course** by Alexey Grigorev.  
The goal is to build a practical AI Agent capable of autonomously working with documentation, context, and retrieval.

I am completing this bootcamp in parallel with the **AI Dev Tools Zoomcamp** to reinforce applied skills and concepts.

## ðŸš€ Bootcamp Overview (7 Days)

### Day 1: Ingest and Index Your Data
- Fetch GitHub repository content
- Extract documentation
- Prepare data for search
- **Deliverable:** Project data downloaded and ready to be processed

### Day 2: Intelligent Processing for Data
- Cut and chunk your data for better search
- Split large documents by paragraphs and sections
- Apply AI-assisted intelligent chunking
- **Deliverable:** Project data fully processed and ready for search

### Day 3: Add Search
- Implement lexical search for exact matches and keywords
- Implement semantic search using embeddings
- Combine into a hybrid search system
- **Deliverable:** Working search system you can query about your project

### Day 4: Agents and Tools
- Add reasoning and function calling for intelligent responses
- Use OpenAI API for function calling
- Define tools for the agent
- Build agents with Pydantic AI
- **Deliverable:** Agent capable of answering questions about your database

### Day 5: Offline Evaluation and Testing
- Create evaluation datasets for your specific use case
- Evaluate your search
- Use AI to evaluate your agent
- **Deliverable:** Thoroughly tested agent with performance metrics

### Day 6: Publish Your Agent
- Create a UI for your agent (e.g., Streamlit)
- Publish it online
- **Deliverable:** Live, accessible AI agent available on the web

### Day 7: Share Results & Peer Review
- Write a project README
- Create demo videos and screenshots
- Review three fellow participants' projects
- Add project to portfolio with technical details
- Share progress on social media
- Receive signed certificate of completion
- **Deliverable:** Portfolio-ready project with peer feedback and certificate
